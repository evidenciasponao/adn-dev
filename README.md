## Overview

ADN DEV Cloud is a modern developer portfolio designed to present software engineering projects, technical skills and cloud deployment experience through a clean and responsive user experience.

The platform serves as a central hub for recruiters, clients and collaborators interested in reviewing completed projects and technical capabilities.

---

## Problem Solved

A developer needs more than a resume.

Traditional CVs often fail to demonstrate technical skills, project architecture and deployment experience.

ADN DEV Cloud was built to provide:

* Professional online presence
* Centralized project showcase
* Technical skills presentation
* Cloud deployment demonstration
* Direct contact channel

## Local Development

### Prerequisites

* Node.js 20+
* Angular CLI

### Installation

```bash
git clone https://github.com/YOUR_USERNAME/adn-dev-cloud.git

cd adn-dev-cloud

npm install
```

### Run Development Server

```bash
ng serve
```

Open:

```text
http://localhost:4200
```
